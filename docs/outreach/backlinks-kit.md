# Backlinks & Citations — paste-ready kit

The link race is winnable: only one competitor (rhillane.com) has real link authority; everyone else ranking is tiny. You don't need many links — you need **consistent citations** (same NAP everywhere) + a few relevant ones. Work top-down; Tier 0 and Tier 4 are the highest ROI.

## 🔑 Canonical NAP block — paste this IDENTICALLY everywhere (consistency is the signal)
```
WeReact agency
Marrakech 40000, Morocco
+212 602-258009
hello@wereact.agency
https://www.wereact.agency
```
- **Business name:** always `WeReact agency` — never add keywords.
- No street address (service-area business) → where a directory demands a street, use city-level only or mark "service area / no storefront".
- **Short description (use where space is tight):**
  > Bilingual (FR/EN) web design studio in Marrakech. Fast, SEO-ready business, e-commerce, and direct-booking websites, delivered in 2–3 days. Showcase sites from 2,000 MAD.
- **Long description:** reuse the EN/FR blocks from `gbp-kit.md`.
- **Categories to select:** Website designer / Web design / Digital agency / Marketing agency.

## Tier 4 — ⭐ The easiest, most relevant links you have: client footer credits
A "Site par WeReact →" link in the footer of the 3 sites you built is the most relevant, natural backlink possible — same industry, real work. Ask the owners. If you built the sites, you may be able to add it yourself.

**Request (FR):**

**Request (EN):**
