# Target-Finding Worksheet — how to build your outreach lists

## Read this first (it will save your reputation)

I tried to hand you a ready list of "riads/businesses with no website." I
**deliberately didn't**, and here's why: web search surfaces the businesses that
*rank* — which are exactly the ones that **already have a website**. When I
verified real candidates (Riad Atay, Riad Shajara, Le Petit Cornichon), every
single one turned out to have its own site. If I'd listed them as "no website"
prospects, your opening message —*"I noticed you don't have a direct-booking
site"*— would be instantly, embarrassingly wrong.

**The targets you want are invisible to search by definition.** You find them by
browsing the platforms directly and verifying each one. It takes ~30 min to build
a week's list. This worksheet is that process. Log everything in
`outreach-tracker.csv`.

## The 10-second verification test (use on EVERY candidate)

Before adding anyone to the list, Google exactly this:
> **`"[business name]" Marrakech site officiel`**  (or `official website`)

- If the top results are their **own domain** (not booking.com / tripadvisor /
  instagram / facebook / a directory) → they HAVE a site → **not a target**
  (unless the site is clearly broken/outdated — that's List C).
- If the only results are **OTA pages, social profiles, and directories** → **no
  real site → TARGET.** ✅

This one check is what separates a credible outreach message from an
embarrassing one. Never skip it.

## LIST A — Riads & guesthouses paying OTA commission

**Where:** booking.com → Marrakech → filter **Property type: Riad**, sort by
**review score / number of reviews**. Target the sweet spot: **30–300 reviews**
(established enough to have revenue, small enough to lack a team/agency).

**For each candidate:**
1. Open its Booking page → note the exact name.
2. Run the 10-second test.
3. If no own site → it's a target. Find the contact: Google Maps listing often
   shows a phone; the riad's Instagram (search `[name] marrakech instagram`)
   usually has a WhatsApp/DM route.
4. Log: name · #reviews · "no site, OTA+IG only" · IG handle / phone · hook.

**Hook (FR):** *"J'ai vu [Riad X] sur Booking — superbes avis ! Je n'ai pas
trouvé votre site de réservation directe. Chaque réservation Booking vous coûte
15–25% de commission…"* (full script in `../client-acquisition-no-ads.md` §1).

**Also try:** tripadvisor.com Marrakech → Speciality lodging; Airbnb "hotels"
category. Same verification.

## LIST B — Instagram-first businesses with no website

**Where:** Instagram search these hashtags/locations, filter to accounts posting
weekly, **1k–30k followers**, **no real website in bio** (empty, or a
linktr.ee/wa.me link):
`#marrakechrestaurant` · `#spamarrakech` · `#hammammarrakech` ·
`#quadmarrakech` · `#buggymarrakech` · `#marrakechshopping` ·
`#beautymarrakech` · location tag "Marrakech, Morocco".

**For each:** run the 10-second test on the business name. IG-only + no site =
target. Contact route = IG DM or the WhatsApp number in their bio.

**Hook (FR):** *"Votre page Instagram est très pro 👌 mais quand on vous cherche
sur Google, il n'y a rien à réserver…"*

## LIST C — Businesses with outdated / broken sites

**Where:** Google these, then go to **page 2–3** (page 1 = the competitors who
did SEO right; page 2–3 = the neglected sites):
`restaurant marrakech` · `location voiture marrakech` · `traiteur marrakech` ·
`architecte marrakech` · `clinique dentaire marrakech` · `avocat marrakech` ·
`agence immobilière marrakech`.

**Open each site and note a CONCRETE problem** (this becomes your hook):
- Not mobile-friendly (pinch-to-zoom on your phone)
- `http://` with no padlock (no SSL)
- Broken pages / images, "under construction", Flash relics
- Free-tier banner (wix.com/ subdomain, blogspot)
- Copyright footer says 2019 or older

**Hook:** screenshot their site next to one of yours → *"J'ai remarqué que votre
site ne s'affiche pas bien sur mobile — voici une comparaison…"* The image sells
it.

## Weekly quota & qualification checklist

Build **10 verified targets/day** (mix of the 3 lists). Before logging each, tick:
- [ ] Ran the 10-second verification test (no own site, OR site is genuinely broken)
- [ ] Business is real & active (recent reviews/posts)
- [ ] I have a contact route (IG handle, WhatsApp, or phone)
- [ ] I wrote a *specific* first-line hook (their name + what I actually noticed)

50 verified targets/week → ~5–8 replies → ~2–3 real conversations → your first
client. Log all of it in `outreach-tracker.csv` and let the numbers guide you.
