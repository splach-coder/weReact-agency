# Just Luxury Transfers reference archive

This archive contains a screenshot and metadata record for every route listed in the source sitemap.

- `manifest.json` contains the complete route inventory and capture status.
- `section-index.json` contains detected headings and section records.
- `pages/<locale>/<family>/<slug>/` contains the screenshot(s) and `page.json` for each route.
- Long pages are split into `page-01.jpg`, `page-02.jpg`, and so on.

The archive currently covers 179 routes across English, French, Spanish, Italian, and German.
